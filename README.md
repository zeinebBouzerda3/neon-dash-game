🎮 Aperçu

NEON DASH est un jeu de runner infini en 3 voies développé en HTML5 Canvas pur.

Aucun framework
Aucun serveur
Fonctionne hors-ligne

👉 Évite les obstacles, collecte des coins, active ton boost et bats ton meilleur score dans un univers cyberpunk néon dynamique.

✨ Fonctionnalités
🚀 Feature	💡 Détail
🏃 Runner infini	Vitesse qui augmente progressivement
🛤️ 3 voies	Déplacement fluide avec interpolation
🦘 Saut & Glissade	Physique arcade dynamique
❤️ 3 vies	Invincibilité temporaire après collision
⚡ Boost	Barre à remplir avec les coins
🛡️ Power-ups	Bouclier + recharge boost
🌆 Univers cyberpunk	Effets néon, grille perspective, particules
📱 Mobile friendly	Contrôles tactiles (swipe)
💾 Sauvegarde	Meilleur score via localStorage
🕹️ Contrôles
⌨️ Clavier
Touche	Action
← / →	Changer de voie
↑	Sauter
↓	Glisser
Espace	Activer le boost
📱 Mobile / Tactile
Geste	Action
Swipe gauche / droite	Changer de voie
Swipe haut	Sauter
Swipe bas	Glisser
🚀 Lancer le jeu en local
🔹 Méthode simple
git clone https://github.com/zeinebBouzerda3/neon-dash-game.git
cd neon-dash-game
start index.html   # Windows
🔹 Serveur local (optionnel)
# Python 3
python -m http.server 8000

👉 Ouvre ensuite : http://localhost:8000

🗂️ Structure du projet
neon-dash-game/

🧠 Stack technique
🎨 Rendu : HTML5 Canvas API
⚙️ Physique : moteur custom (gravité, collisions AABB)
🎞️ Animations : requestAnimationFrame
💾 Stockage : localStorage
🌍 Compatibilité : Chrome, Firefox, Safari, Edge
